  <name>Task 1: Implement Core Monte Carlo Pi Logic in pi_MPI.c</name>
  <files>Problem_4/pi_MPI.c</files>
  <behavior>
    - Test 1: Program initializes MPI correctly and gets rank/size.
    - Test 2: Each process generates random (x, y) coordinates within [0, 1) range.
    - Test 3: Each process accurately counts points falling inside the unit circle (x² + y² < 1).
    - Test 4: Local `points_in_circle` and `total_points` are calculated per process.
  </behavior>
  <action>
    Modify `Problem_4/pi_MPI.c` to:
    1. Include necessary headers (`mpi.h`, `stdio.h`, `stdlib.h`, `time.h`, `math.h`).
    2. Initialize MPI (`MPI_Init`) and finalize (`MPI_Finalize`).
    3. Get `world_rank` and `world_size`.
    4. Seed the random number generator using `srand48(time(NULL) + world_rank)` for process-specific seeds.
    5. Define `num_iterations_per_process` (e.g., 1,000,000 or a value that can be easily adjusted).
    6. Implement a loop to generate `num_iterations_per_process` random (x, y) points using `drand48()`.
    7. For each point, check if `x*x + y*y < 1.0` and increment `local_in_circle_count` if true.
    8. Store `local_in_circle_count` and `num_iterations_per_process` for later reduction.
  </action>

      grep "Process 1 local in circle" mpi_output.txt
    </automated>
  </verify>
  <done>
    `pi_MPI.c` compiles, and when run with MPI, each process prints its locally calculated `in_circle_count`.
  </done>
