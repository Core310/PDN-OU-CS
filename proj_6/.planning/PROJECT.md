# Project 6: MapReduce and Network Programming

## Overview
This project involves implementing a MapReduce job to find the longest words starting with each letter of the alphabet using the `mrjob` library. Additionally, it requires creating TCP and UDP client-server applications for performing remote multiplication on comma-separated numbers.
