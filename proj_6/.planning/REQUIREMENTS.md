# Requirements - Project 6

## Problem 1: MapReduce
- **R1.1:** Use `mrjob` library to implement the MapReduce job.
- **R1.2:** Find the longest word(s) for each letter 'a' through 'z'.
- **R1.3:** Handle ties: If multiple words have the same maximum length for a letter, all must be listed.
- **R1.4:** Output format: Key is the starting letter, Value is a list of longest words.
- **R1.5:** Execution: `python find_longest_words_MRJob.py 1342-0.txt`.

## Problem 2: Network Programming
- **R2.1:** Implement TCP Server and Client (`TCPServer_multiply.py`, `TCPClient_multiply.py`).
- **R2.2:** Implement UDP Server and Client (`UDPServer_multiply.py`, `UDPClient_multiply.py`).
- **R2.3:** Client must prompt for comma-separated numbers.
- **R2.4:** Server must calculate the product of received numbers.
- **R2.5:** Server must return the product or "Invalid input".
- **R2.6:** Use IP `127.0.0.1`.
