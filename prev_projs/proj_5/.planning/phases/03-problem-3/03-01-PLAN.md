<task type="auto" tdd="false">
  <name>Task 2: Implement Tree-Based Merge</name>
  <files>
    - Problem_3/merge_sort_MPI.c
  </files>
  <action>
    1.  Implement a loop that iterates `log2(comm_size)` times.
    2.  Inside the loop, processes will be paired up. If a process's rank is in the upper half of the current group of processes, it will send its sorted sub-array to its partner in the lower half.
    3.  The receiving process will then merge its own sorted sub-array with the one it just received, creating a new, larger sorted sub-array.
    4.  This process repeats, with the group of active processes halving in each iteration, until process 0 has the final, fully sorted array.
  </action>
  <verify>
    <automated>
      Compile and run the code. Use a debugger or print statements to verify that process 0 has the complete, sorted array at the end of the merge phase.
    </automated>
  </verify>
  <done>
    Process 0 has the final, sorted array.
  </done>
</task>
